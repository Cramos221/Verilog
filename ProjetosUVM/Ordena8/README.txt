# README: Teste UVM para Verificação de Ordenação de 8 Números

## Descrição

Este projeto consiste em um teste UVM (Universal Verification Methodology) em desenvolvimento, destinado à verificação de um DUT (Design Under Test) cujo objetivo é realizar a ordenação de uma lista de 8 números. O teste foi projetado para validar se o DUT é capaz de ordenar corretamente a entrada fornecida.
